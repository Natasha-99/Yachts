## Яхты "Black Sea Yachts"

ИБ разработана на платформе «1С:Предприятие 8», которая представляет собой автоматизированную систему управления продажами и учетом заказов для верфи Black Sea Yachts.

**В базе реализовано:**
* ***Справочники:*** Каталог продукции (яхты и лодки с характеристиками: тип, материал, цвет, наличие мачты), совместимое дополнительное оборудование, контрагенты (клиенты-физлица и организации с паспортными данными и сканами документов), партнеры и субподрядчики. Предусмотрено хранение истории изменения цен и поддержка списка снятых с производства моделей.
* ***Документооборот:*** Оформляются «Заказы клиентов» с уникальными номерами и привязкой к менеджеру. Реализован механизм формирования комплекта документов: Договор (блокируется для редактирования после подписания), Счет на предоплату, Спецификация лодки/яхты и Список дополнительного оборудования. Все документы формируются автоматически на основе данных заказа и отправляются клиенту на email.
* ***Финансовый учет и контроль:*** Система контролирует график платежей: обязательная предоплата не менее 30%, фиксация факта оплаты бухгалтером с привязкой к заказу.
* Реализован ***расчет НДС*** с индивидуальными ставками для разных товаров и услуг. Встроена проверка личности клиента перед подписанием договора.
* ***Аналитика и отчетность:*** Сформированы отчеты для выявления премиальных клиентов (частота заказов, сумма самых дорогих покупок). Хранится полная история взаимодействий, адресов доставки и подтверждений получения товара (дата, время, получатель).
* ***Интерфейс:*** Система адаптирована для пользователей без IT-подготовки (менеджеров и строителей). Интерфейс разделен на логичные блоки («Каталог», «Заказы», «Клиенты», «Администрирование»), предусмотрены удобные формы ввода данных с защитой от ошибок и разграничение прав доступа (отдельный интерфейс для Администратора по управлению ценами и номенклатурой).

PS:
Паролей у ролей нет, чтобы войти выберите Администратор и нажмите enter.
